Buttons

The Button component consists of different variants to suit a number of use cases within the JobAdder system. Each button may have different states and comes in 2 different sizes, which may include icons on either side.

Primary

The Primary buttons are used as the main call to action of a form or any single flow within a screen. It should only appear once and can be paired with a Default or Text button to display a sense of hierarchy within the actions.

<button nz-button nzType="primary">
    Primary Button
</button>

Default

Default buttons are used to illustrate an actionable item within a screen which does not necessarily has to be the main call to action.

<button nz-button nzType="default">
    Default Button
</button>

Text

Text buttons are normally used to accompany Primary buttons within a certain flow.

<button nz-button nzType="text">
    Text Button
</button>

Danger

The Primary, Default & Text buttons may all have a Danger variant which signifies a destructible action. Choose the appropriate button type depending on the severity of the action.

<button nz-button nzType="primary" nzDanger=”true”>
    Delete Candidate
</button>

Status

Status buttons are JobAdder specific buttons which displays the current Status of a certain record. It can be also paired with a Selection List which may update the button’s Status value.

<button nz-button nzType="primary" nzDanger=”true”>
    Delete Candidate
</button>

Link

Use a link button to navigate to another page. These should open in the same window unless information may be lost (for example, when someone is filling out a form), or when the destination is an external site (for example, a knowledge base article).

Candidate Record 
<a nz-button nzType="link">
    Candidate Record
</a>
